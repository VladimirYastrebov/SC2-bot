# StarCraft II Protoss Bot

A Protoss bot for StarCraft II built with Python and the burnysc2 library.

## Features

- Builds Probes and Pylons
- Produces Zealots, Stalkers and Immortals
- Attacks with a deathball army!
- Uses Chrono Boost on Nexus
- Can win against SC2 AI on Easy mode

## Requirements

- Python 3.8+
- StarCraft II installed
- burnysc2 library

## How to Run

```bash
pip install burnysc2
python run.py
